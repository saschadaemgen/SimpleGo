SimpleGo Documentation

Welcome to the official SimpleGo technical documentation.

SimpleGo is the world's first native C implementation of the SimpleX Messaging Protocol (SMP) on bare-metal embedded hardware. No smartphone. No Android. No baseband processor.

Where to start​

If you want to flash a device, go to Getting Started → Quick Start.

If you want to understand the protocol implementation, start with SMP in C → Overview.

If you want to contribute code, read Contributing → Build first.

If you want to understand why this exists, read Architecture → Encryption Layers.
