# SimpleGo Roadmap

## Phase 1 — Transport + SMP Core ✅
- TLS 1.3 connection to SMP servers
- SMP command encoding/decoding
- Queue creation and subscription
- Message receiving and acknowledgment

## Phase 2 — E2E Encryption ✅
- NaCl server-recipient decryption
- E2E sender authentication
- X3DH key agreement
- Double Ratchet with header encryption
- Complete receive chain (contact queue)

## Phase 3 — Bidirectional Messaging 🔨
- Reply Queue implementation
- AgentConnInfo exchange
- Full duplex connections
- Text message sending

## Phase 4 — Extended Features 📋
- Delivery receipts
- Message editing and deletion
- File transfer (XFTP)
- Multiple connections

## Phase 5 — Hardware Product 📋
- Custom PCB design
- Battery management
- Enclosure design
- Multi-device support (HAL architecture)

## Phase 6 — Advanced 📋
- Group messaging
- Post-quantum KEM (SNTRUP761)
- Voice messages
- Push notifications
- Private routing
